Bengaluru House Price Prediction
Project Overview

This project is a Machine Learning-based web application that predicts house prices in Bengaluru based on user inputs such as area (square feet), location, number of bedrooms (BHK), and bathrooms.
It demonstrates the complete ML workflow including data preprocessing, feature engineering, model building, and deployment using a web interface.

Features

Predict house prices based on user inputs
Clean and interactive user interface
Real-time predictions using trained ML model
End-to-end pipeline from data to deployment

Tech Stack

Programming Language: Python
Libraries: NumPy, Pandas, Scikit-learn
Backend: Flask
Frontend: HTML, CSS, JavaScript

Machine Learning Workflow
Data Cleaning and Preprocessing
Feature Engineering (handling categorical data like location)
Model Training using Linear Regression
Model Evaluation and Optimization

How to Run the Project
1️⃣ Clone the repository

2️⃣ Navigate to project folder
cd BHP
3️⃣ Install dependencies
pip install -r requirements.txt
